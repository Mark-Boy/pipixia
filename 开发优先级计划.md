## 技术栈总览

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | Next.js 16 + TypeScript | App Router + Server Components |
| **样式** | Tailwind CSS v4 + Shadcn/UI | 原子化样式 + 高质量组件 |
| **状态** | Zustand + React Query | 全局状态 + API 缓存 |
| **图表** | Recharts | 数据可视化 |
| **后端** | Python + FastAPI | 高性能异步 API |
| **数据库** | PostgreSQL 16 | 持久化存储 + JSONB |
| **缓存/队列** | Redis 7 + Celery 5 | 任务队列 + 分布式锁 |
| **AI 编排** | LangChain + LangGraph | 翻译/风控/SEO 有向图工作流 |
| **LLM** | DashScope（通义千问） | 云端 API，初期使用 |
| **对象存储** | MinIO（开发）/ OSS（生产） | 图片存储 |
| **OCR** | PaddleOCR | 图片文字提取 |
| **反爬** | Playwright + 代理池 | 1688/拼多多数据采集 |
| **部署** | Docker + Docker Compose | 容器化部署 |
| **监控** | Prometheus + Grafana | 指标可视化 |
