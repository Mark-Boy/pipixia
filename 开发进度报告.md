## 📊 API 端点

### 店铺
- `GET /api/v1/shopee/shop-info` — 店铺信息
- `POST /api/v1/shopee/authorize` — OAuth 授权
- `POST /api/v1/shopee/oauth/callback` — 回调处理

### 商品
- `GET /api/v1/shopee/products` — 商品列表
- `GET /api/v1/shopee/listings` — 上架记录

### 同步
- `POST /api/v1/shopee/list` — **自动采集 + 上传**
- `POST /api/v1/shopee/sync-stock` — 库存同步
- `POST /api/v1/shopee/sync-products` — 商品列表同步
- `GET /api/v1/shopee/status` — 同步状态

### 调度器
- `POST /api/v1/shopee/start-scheduler` — 启动
- `GET /api/v1/shopee/scheduler/status` — 状态
- `POST /api/v1/shopee/scheduler/stop` — 停止

## 🔄 每分钟自动汇报

调度器每分钟执行:
1. 检查待上架商品 → 自动上架
2. 同步库存到 Shopee
3. 每 5 分钟同步商品列表
4. 输出进度报告到控制台
5. 可选: 发送到 Telegram
