## 快速启动

```bash
# 1. 启动基础设施
docker compose up -d postgres redis minio minio-init

# 2. 运行迁移
make migrate

# 3. 从 SQLite 迁移数据（如有）
make pg-migrate

# 4. 创建管理员
make create-admin

# 5. 启动 API
make up
```
